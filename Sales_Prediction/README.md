# CodeAlpha - Sales Prediction using Python
## Project Overview
This project predicts product sales based on advertising expenditures using Machine Learning. The model analyzes the relationship between advertising budgets and sales performance to estimate future sales.
## Objective
The objective of this project is to build a predictive model that can estimate sales using advertising data and evaluate its performance using regression metrics.
## Dataset Description
The dataset contains advertising expenditure data across different marketing channels such as:
* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales
## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
## Machine Learning Algorithm
* Linear Regression
